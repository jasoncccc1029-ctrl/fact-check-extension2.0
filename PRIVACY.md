# 事實查核助手（瀏覽器擴充功能）隱私權說明

**最後更新：2026 年 4 月 17 日**

本說明描述擴充功能「**事實查核助手**」（以下稱**本擴充**）如何處理您的資料。

---

## 1. 資料控制者與聯絡方式

- **資料控制者**：Jason（個人開發者）
- **聯絡方式**：請透過 Chrome Web Store 商店頁面的「聯絡開發人員」功能提出
- 若您對本擴充之隱私實務有疑問，請透過上述方式聯繫。

---

## 2. 本擴充會取得或處理哪些資料？

### 2.1 您主動選取或輸入的內容

- **選取文字**：當您劃選網頁文字並使用查核功能（或快捷鍵）時，該段文字會由本擴充送往**查核相關服務**以取得結果（詳見第 3 節）。  
- **頁面正文匯出**：若您使用「匯出頁面報告」類功能，擴充會從目前頁面擷取一段限定長度內的文字，用於產生報告；擷取範圍與上限可由您在本機設定中調整。

### 2.2 儲存在您瀏覽器本機的資料（Google Chrome `storage`）

下列資料僅存於您的裝置本機，**本擴充作者預設無法遠端讀取**（除非您另行安裝同步或備份工具且其規範另計）：

| 類別 | 說明 |
|------|------|
| **查核歷史** | 例如最近一次查詢字串、結果筆數與時間等摘要，供擴充視窗顯示；您可於介面中清除。 |
| **側欄狀態** | 例如側欄最近查核紀錄列表、淺色／深色主題偏好。 |
| **觸發鈕位置偏好** | 例如查核觸發鈕是否固定在螢幕角落。 |
| **選用：Gemini API 金鑰與模型** | 若您自行在擴充設定中填入 Google Gemini API 金鑰與模型名稱，該金鑰存於 `chrome.storage.local`，**僅用於在您裝置上發起的 API 請求**；移除金鑰即刪除。 |
| **選用：略過 Gemini** | 是否關閉與 Gemini 相關流程之偏好。 |
| **選用：Gemini 隱私提醒關閉狀態** | 若您在提示中選擇「不再顯示」，會記錄於本機以避免重複彈出。 |
| **媒體分析關鍵字與匯出設定** | 您自訂的關鍵字列表、匯出字數上限等。 |
| **新聞來源降權列表** | 您自訂的網域列表（若功能有啟用）。 |
| **選用：是否顯示可信即時新聞區塊** | 控制側欄是否顯示「即時新聞（可信來源）」區塊。 |
| **選填：查核索引 API 權杖（`apiToken`）** | 若您在擴充選單「查核索引 API（進階）」自行儲存權杖，該值僅存於本機，用於向開發者提供之索引服務識別較高額度客戶；未設定時請求仍可能以匿名層級送出（依伺服器政策與限流為準）。 |
| **工作階段快取** | 查核結果短時快取於 `chrome.storage.session`，以減少重複請求；關閉瀏覽器或工作階段結束後通常會清除。 |

### 2.3 網路傳輸與第三方

當您使用查核相關功能時，可能會將**查核查詢字串**（及完成查核所需之技術性資料，如語系參數）傳送至下列類型之服務：

- **國際查核索引 API**（`fact-check-extension.vercel.app`）：用於查詢公開查核資料庫／索引結果。
- **Cofacts 真的假的**（`api.cofacts.tw`）：用於查詢社群查核與相關內容，資料採 CC BY-SA 4.0 授權。
- **台灣事實查核中心 TFC**（`tfc-taiwan.org.tw`）：用於查詢專業查核報告，僅顯示標題與連結。
- **Google Fact Check Tools API**（`factchecktools.googleapis.com`）：用於查詢國際查核索引。
- **Google News RSS**（`news.google.com`）：用於取得與查核相關之即時新聞標題比對。
- **Google Generative Language API（Gemini）**（`generativelanguage.googleapis.com`）：**僅在您於本機儲存有效 API 金鑰且未關閉相關選項時**，可能將您選取之文字與查核上下文字料送交 Google 以產生輔助判讀。使用 Gemini 免費版時，依 [Google AI 服務條款](https://ai.google.dev/terms)，傳送內容可能用於改善 Google 模型；如需隱私保障請使用 Google AI 付費方案。

本擴充**不以出售使用者資料為業**，亦**不將查核內容用於廣告再行銷**。

## 3. 處理目的與法規依據（摘要）

- **目的**：提供選字查核、查核結果顯示、可選之 AI 輔助判讀、歷史紀錄與介面偏好設定。  
- 若您所屬司法管轄區要求標示法規依據，請依當地法律補充（例如契約履行、正當利益或同意等）。

---

## 4. 資料保存期間

- **本機儲存之歷史與設定**：留存至您清除擴充資料、解除安裝擴充，或於擴充介面／瀏覽器設定中刪除為止。  
- **工作階段快取**：屬暫存，期間通常限於單次瀏覽工作階段。  
- **第三方服務**（Cofacts、Google、您部署之 API）：其保留政策請見各該服務之隱私權政策。

---

## 5. 您的權利與控制方式

- 您得隨時在擴充設定中**移除 Gemini API 金鑰**、**清除歷史紀錄**，或透過 Chrome「擴充功能 → 事實查核助手 → 移除」，以刪除本機儲存之資料。  
- 若您所在的法域賦予查閱、更正、刪除、限制處理、資料可攜等權利，請透過第 1 節聯絡方式提出；涉及第三方服務者，依法亦可能須直接向該第三方行使。

---

## 6. 兒童隱私

本擴充并非以兒童為主要對象。若您認為我們不慎處理了兒童資料，請與我們聯繫以便刪除。
